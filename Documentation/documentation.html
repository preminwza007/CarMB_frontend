Self-Drive Frontend Documentation

Version 1.0.0 - Updated with latest dependencies (React 19, Bootstrap 5, Chart.js 4)

Overview

A modern React-based dashboard for autonomous vehicle control and monitoring, featuring real-time camera feeds, telemetry data, and interactive controls.

Technology Stack

  • React 19.2.0 - Latest React with concurrent features
  • Bootstrap 5.3.8 - Modern CSS framework
  • Chart.js 4.5.1 - Data visualization
  • Leaflet 5.0.0 - Interactive maps
  • FontAwesome 7.1.0 - Icon library

Getting Started

npm install
npm start

WebSocket Configuration

Update the WebSocket URL in src/views/examples/Maps.js:

const ws = new WebSocket("ws://YOUR_SERVER:PORT/ws/browser");

Key Components

  • CameraStream - Live video feed display
  • MapComponent - Navigation and route planning
  • IconCards - Control panel for vehicle functions
  • Speedometer - Real-time speed display
  • ClimateCard - Climate control interface

Build & Deploy

npm run build
serve -s build
